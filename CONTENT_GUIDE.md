## 🃏 Atributos de Cartas

### Jogadores de Campo

Cada jogador tem 6 atributos principais:

1. **Chute (0-100):** Poder do chute, importante para gols
2. **Ataque (0-100):** Capacidade de ataque, importante para criar oportunidades
3. **Defesa (0-100):** Capacidade defensiva, importante para bloquear ataques
4. **Força (0-100):** Força física, afeta confrontos
5. **Salto (0-100):** Altura do salto, importante para cabeceios
6. **Estamina (0-100):** Resistência, importante para manter performance

### Goleiros

Cada goleiro tem 3 atributos:

1. **Habilidade com Mãos (0-100):** Capacidade de defesa com as mãos
2. **Agilidade (0-100):** Velocidade de reação
3. **Salto (0-100):** Altura do salto para defesas aéreas

## ⚙️ Atributos de Equipamentos

Equipamentos podem aumentar os seguintes atributos:

- Chute: +2 a +15
- Ataque: +2 a +12
- Defesa: +2 a +15
- Força: +2 a +12
- Salto: +1 a +10
- Estamina: +1 a +12
- Habilidade com Mãos: +2 a +15
- Agilidade: +1 a +10

## 🎁 Sistema de Baús

### Como Funcionam

1. **Compra:** Jogador compra baú com moedas do jogo
2. **Abertura:** Baú começa a abrir (tempo varia por qualidade)
3. **Recompensas:** Após abrir, jogador recebe:
   - Cartas aleatórias
   - Moedas do jogo
   - Moedas especiais (com probabilidade)

### Probabilidades de Moedas Especiais

- Comum: 5%
- Incomum: 15%
- Raro: 35%
- Épico: 65%
- Lendário: 95%

## 💰 Economia do Jogo

### Moedas do Jogo

- **Obtenção:**
  - Vitórias em partidas: 50-200 moedas
  - Baús: 100-5000 moedas
  - Missões diárias: 100-500 moedas

- **Uso:**
  - Comprar baús: 500-10000 moedas
  - Comprar equipamentos: 300-10000 moedas
  - Fazer upgrade de cartas: 100-1000 moedas por nível

### Moedas Especiais

- **Obtenção:**
  - Baús (com probabilidade): 0-150 moedas
  - Compra com dinheiro real: 500-10000 moedas
  - Eventos especiais: 50-500 moedas

- **Uso:**
  - Comprar cartas premium: 100-500 moedas
  - Comprar equipamentos premium: 50-200 moedas
  - Acelerar abertura de baús: 10-50 moedas

## 🎯 Progressão de Cartas

### Níveis

Cada carta pode ter até 10 níveis:

- **Nível 1:** Base
- **Nível 2-10:** Upgrades progressivos

### Requisitos para Upgrade

Para fazer upgrade de uma carta do nível N para N+1:

- **Cópias:** N cópias da carta
- **Moedas:** 100 × N moedas do jogo

Exemplo: Para fazer upgrade de nível 1 para 2:
- Precisa de 1 cópia adicional
- Custa 100 moedas

### Bônus por Nível

Cada nível aumenta todos os atributos em ~5%:

- Nível 1: 100% dos atributos
- Nível 2: 105% dos atributos
- Nível 3: 110% dos atributos
- ...
- Nível 10: 145% dos atributos

## 🔧 Equipamentos

### Slots

Cada carta pode equipar até 3 itens:

- Slot 1: Chuteira
- Slot 2: Luvas ou Protetor
- Slot 3: Acessório (Fita, Faixa, Meia, Caneleira)

### Evolução de Equipamentos

Equipamentos também podem evoluir:

- **Nível 1-5:** Bônus progressivos
- **Custo:** 50 × N moedas (sem precisar de cópias)

## 📈 Balanceamento

### Distribuição de Qualidades

- **Comum:** 50% de chance em baús comuns
- **Incomum:** 40% em baús incomum, 30% em raros
- **Raro:** 20% em baús raros, 40% em épicos
- **Épico:** 30% em baús épicos, 50% em lendários
- **Lendário:** 20% em baús lendários

### Poder Relativo

- Comum: 100 poder base
- Incomum: 125 poder base
- Raro: 150 poder base
- Épico: 175 poder base
- Lendário: 200 poder base

## 🚀 Adicionar Novo Conteúdo

### Adicionar Carta

1. Editar `src/data/cards.ts`
2. Adicionar objeto CardData
3. Executar `npm run seed`

### Adicionar Equipamento

1. Editar `src/data/equipment.ts`
2. Adicionar objeto EquipmentData
3. Executar `npm run seed`

### Adicionar Baú

1. Editar `src/data/chests.ts`
2. Adicionar objeto ChestData
3. Executar `npm run seed`
